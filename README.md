# Plant Diary
나의 일기에서 묻어나온 감정을 AI에게 분석시켜 나온 결과에 따라 자라나는 식물을 관리하는 앱

# 주요기능
- 로그인, 아웃 기능
- 일기 작성 기능
- 감정 분석 기능
- 일기, 감정 분석 결과 날짜별로 관리 기능
- 식물 관리 기능

# 작동환경
- (Xcode: 15.4)
- (iOS: 17.5)
